# ntv2-pt


The NTv2 (National Transformation version 2) is a standard binary grid shift (.GSB) file format.

Downloaded from:
    http://www.dgterritorio.pt/cartografia_e_geodesia/geodesia/transformacao_de_coordenadas/grelhas_em_ntv2/
